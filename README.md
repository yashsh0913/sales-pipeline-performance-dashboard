# CRM Conversion Analysis & Insights

## Overview

This project focuses on analysing CRM conversion performance across multiple dimensions to uncover patterns, inefficiencies, and opportunities for improving lead-to-customer conversion.

---

## Business Problem

While funnel visibility highlights drop-offs, it often fails to explain *why* conversions succeed or fail. This leads to:

* Limited understanding of conversion drivers
* Ineffective targeting and prioritisation
* Suboptimal pipeline performance

This project addresses these gaps through deeper conversion-focused analysis.

---

## Approach

* Structured CRM dataset using SQL for analysis
* Analysed conversion rates across segments (source, stage, timeline)
* Performed cohort-style analysis to track lead behaviour over time
* Identified high-performing vs underperforming segments
* Built analytical views to support decision-making

---

## Key Insights

* Certain lead sources significantly outperform others in conversion
* Conversion likelihood drops sharply beyond specific time thresholds
* Mid-funnel inefficiencies impact final deal closure rates

---

## Impact

* Identified actionable levers to improve conversion efficiency
* Enabled targeted interventions across high-impact segments
* Supported data-driven prioritisation of leads and resources
